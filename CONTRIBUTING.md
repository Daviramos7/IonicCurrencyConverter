Como Contribuir (Fluxo de Trabalho)
Nós usamos um fluxo baseado em Pull Requests. Nunca envie commits diretamente para a branch main.

1. Crie sua Branch
Sempre comece criando uma nova branch a partir da main, que deve estar sempre atualizada.

2. Faça suas Alterações e Commits
Trabalhe na sua branch, fazendo commits pequenos e com mensagens claras.

3. Envie sua Branch (Push)
Quando terminar sua tarefa (ou quiser salvar seu progresso), envie sua branch para o repositório remoto:

4. Abra um Pull Request (PR)
Vá até a página do repositório no GitHub. Você verá um aviso para abrir um Pull Request (PR) da sua branch para a branch main.

Dê um título claro ao PR.

Descreva o que foi feito.

Adicione os outros membros da equipe como "Revisores".

Após a revisão e aprovação, o Pull Request será "juntado" (merge) à branch main e sua contribuição estará oficialmente no projeto.
